    • write

      Declaration

      auto write(X, Y, Args...)(X tx, Y ty, Args args) if (isNumeric!X && isNumeric!Y && args.length >= 1);

      Write to the internal buffer. Later on print() must be used to display onto the window

      Parameters

      X tx

      The x-position of where to write in the internal buffer

      Y ty

      The y-position of where to write in the internal buffer

      Args args

      What is to be written. C

      Example:

      1. // will display on screen "hello world 42[1337, 1001, 1]" in varied colors window.write(3, 5, "hello ", fg(Color.green), 'w', Cell('o', fg(Color.red)), "rld ", 42, [1337, 1001, 1]); window.print();

      Example:

      1. // will display "lorem" at position [x: 4, y: 3] real x = 4; float y = 3.84; window.write(x, y, "lorem"); // `y` is trucated/converted to `int` window.print();

      Note: Does not directly write to the window, changes will only be visible after window.print();

      Note: **(Authors note)** When I got into D, I thought template functions were odd. So for beginners, the method can be viewed as auto write(int x, int y, arg1, arg2, ..., argN). The X and Yarguments must be numbers, and you can enter how many arguments as you want.
